# plearnet-vue

## CMD
```
npm i -g @vue/cli
<!-- cd到專案底下然後npm install -->
vue cli
```

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### json-server
```
npm install -g json-server
json-server --watch db.json
```

### three.js
```
npm install three
<!-- 擴充點擊功能 -->
npm install three.interactive
```
